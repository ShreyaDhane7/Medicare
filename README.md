# 🏥 Medicare - AI Disease Prediction & Medicine Recommendation System

An AI-powered healthcare web application built using **Python**, **Flask**, and **Machine Learning**. The application predicts diseases based on user-entered symptoms and recommends appropriate medicines from a predefined database.

---

## 📌 Features

- 🤖 Predicts diseases using a trained Machine Learning model
- 💊 Recommends medicines based on the predicted disease
- 🌐 Interactive web interface built with Flask
- ⚡ Fast and easy symptom-based prediction
- 📱 Clean and user-friendly interface

---

## 🛠️ Tech Stack

- **Programming Language:** Python
- **Framework:** Flask
- **Machine Learning:** Scikit-learn
- **Frontend:** HTML, CSS
- **Data Storage:** Pickle (.pkl) files

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/ShreyaDhane7/Medicare.git
cd Medicare
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

---

## 🧠 How It Works

1. Enter the symptoms in the application.
2. The trained machine learning model analyzes the input.
3. The system predicts the most likely disease.
4. Relevant medicine recommendations are displayed.
